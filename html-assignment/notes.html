HTML tags I've learned

<dl> ‐ Description List
Creates a unique type of list which contains a term (<dt>) and a corresponding description(<dd>)
<img> ‐ Image Element
Allows the code to display an image on the webpage. Two attributes are needed for proper use of the <img> tag, <src> and <alt>.
<form> ‐ Form Tag
Through multiple types of inputs, The form tag allows webpages to collect variations of data and send back to the server.
<article> ‐ Article
Used to identify and contain content that could stand alone without the rest of the webpage. Usually used for the primary content of the page.
<cite> ‐ Citation
Used to identify and display citations of a source used on the website.
